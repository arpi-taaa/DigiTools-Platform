🛠️ DigiTools Platform
Welcome to DigiTools, a modern, high-performance dashboard designed to put the best digital utilities right at your fingertips. Whether you're looking to enhance productivity or scale a professional workflow, this platform is built with a focus on speed, clean UI, and a seamless user experience.

✨ Features
Responsive Product Grid: A fluid layout built with Tailwind CSS and DaisyUI that looks great on everything from a massive monitor to your phone.

Dynamic Cart Logic: A fully functional React-based shopping cart. Add items, remove specific products, and watch your total update in real-time.

Intuitive Tab Navigation: Switch between the product gallery and your cart view without annoying page reloads.

Tiered Pricing: Three distinct service levels (Starter, Pro, and Enterprise) designed to grow with your needs.

Instant Notifications: Real-time feedback using react-toastify so you always know when an action is successful.

🚀 Tech Stack
This project was built using the modern "Frontend Speed" stack:

React.js (Functional Components & Hooks)

Vite (For lightning-fast development and optimized builds)

Tailwind CSS (Utility-first styling)

DaisyUI (For polished, accessible components)
